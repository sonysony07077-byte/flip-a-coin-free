## Link Building Strategy

### High-Value Targets

**Education (.edu domains)**
- Teacher resource directories
- University statistics departments
- Online learning platforms
- Educational tool lists

**Developer Communities**
- GitHub Awesome Lists
- Dev.to tool collections
- Stack Overflow answers (where relevant)
- Developer tool directories

**Gaming Communities**
- DnD resource lists
- Board game forums
- RPG tool collections
- Gaming blog roundups

### Link Building Tactics

1. **Resource Page Link Building**
   - Find "best online tools" lists
   - Reach out with value proposition
   - Offer to write guest post

2. **Broken Link Building**
   - Find broken coin flip tool links
   - Offer FlipACoinNow as replacement
   - Use Ahrefs or SEMrush

3. **HARO (Help A Reporter Out)**
   - Respond to queries about:
     - Online tools
     - Decision-making
     - Educational technology
     - Probability/statistics

4. **Directory Submissions**
   - Product Hunt
   - AlternativeTo
   - Slant
   - Capterra (if applicable)
   - G2 (if applicable)

5. **Community Engagement**
   - Answer questions on:
     - Stack Overflow
     - Reddit
     - Quora
     - Discord servers
   - Provide value first, link second
